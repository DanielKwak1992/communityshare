Signup and Account Settings Templates
-------------------------------------
signup_community_partner.html - Top level template for a community partner signing up
signup_educator.html - Top level template for an educator signing up

community_partner_labels.html
signup_account_settings.html - Email and password settings.
signup_common.html - NO LONGER USED.

settings.html - Top level template for editing settings
account_settings.html - Show email, upload picture

personal_settings.html - Personal settings like name, bio, location
more_personal_settings.html - Additional settings that we didn't want to put in signup
